# Medical Q&A Chatbot using MedQuAD Dataset

## Project Overview

This project is a **Medical Question Answering Chatbot** developed using the **MedQuAD dataset**. The chatbot retrieves the most relevant medical answer by comparing the user's question with questions stored in the dataset using semantic similarity.

The project uses **Sentence Transformers** to generate embeddings for medical questions and **cosine similarity** to find the best matching answer.

> **Note:** This chatbot is intended for educational purposes only and should not be used as a substitute for professional medical advice.

---

## Features

* Medical question answering using the MedQuAD dataset.
* Semantic search with Sentence Transformers.
* Retrieval-based chatbot.
* Simple and interactive Streamlit web interface.
* Fast response generation using precomputed embeddings.

## Installation


### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Download the MedQuAD dataset

Download the dataset from:

https://github.com/abachaa/MedQuAD

Extract the folder and place it inside the project directory.

---

## Running the Application

Start the Streamlit application:

```bash
streamlit run app.py
```

The chatbot will open automatically in your browser.

---

## How It Works

1. Loads the MedQuAD XML files.
2. Extracts medical question-answer pairs.
3. Converts all questions into sentence embeddings.
4. Converts the user's query into an embedding.
5. Computes cosine similarity.
6. Returns the answer corresponding to the most similar question.
