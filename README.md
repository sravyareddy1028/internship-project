# AI Internship Projects

## Overview

This repository contains the projects completed as part of my AI Internship. Each task focuses on developing intelligent chatbot systems using Machine Learning, Natural Language Processing (NLP), Retrieval-Augmented Generation (RAG), Large Language Models (LLMs), and Streamlit.

---

# Task 1: Dynamic Knowledge Base Expansion

### Objective

Implement a system for dynamically expanding the chatbot's knowledge base by periodically updating the vector database with new information from specified sources.

### Features

* Automatic knowledge base updates
* Dynamic vector database expansion
* Semantic document retrieval
* Incremental indexing of new documents
* Improved chatbot responses using newly added information

### Expected Outcome

A chatbot capable of automatically incorporating newly available information into its responses over time.

---

# Task 2: Multi-modal AI Assistant

### Objective

Develop a multi-modal AI assistant capable of understanding and reasoning over both text and image inputs.

### Features

* Image understanding and analysis
* Text and image reasoning
* Context-aware conversations
* Evidence-based response generation
* Ambiguity handling
* Intelligent decision making
* Response validation
* Multi-turn conversational memory

### Expected Outcome

An AI assistant capable of extracting information from images, combining it with textual input, maintaining conversational context, and providing accurate, evidence-based responses.

# Task 3: Medical Q&A Chatbot (MedQuAD)

### Objective

Develop a specialized Medical Question Answering chatbot using the MedQuAD dataset.

### Features

* Medical question answering
* Retrieval-based response generation
* Medical entity recognition (Symptoms, Diseases, Treatments)
* Semantic similarity search
* Interactive Streamlit user interface

### Dataset

* MedQuAD Dataset

### Expected Outcome

A medical chatbot capable of retrieving accurate answers to user queries using semantic search techniques.

# Task 4: Domain Expert Chatbot (arXiv)

### Objective

Develop a chatbot that serves as an expert in a specific domain using the arXiv dataset.

### Features

* Scientific paper search
* Research paper summarization
* Concept explanation using open-source LLMs
* Advanced NLP-based information extraction
* Follow-up question handling
* Concept visualization
* Streamlit interface

### Dataset

* arXiv Dataset (Computer Science subset)

### Expected Outcome

A chatbot capable of discussing advanced topics, summarizing research papers, explaining complex concepts, and answering follow-up questions.

# Task 5: Sentiment Analysis Chatbot

### Objective

Integrate sentiment analysis into the chatbot to detect customer emotions and generate appropriate responses.

### Features

* Positive sentiment detection
* Negative sentiment detection
* Neutral sentiment detection
* Emotion-aware chatbot responses
* Improved conversational experience

### Expected Outcome

A chatbot capable of recognizing user emotions and responding appropriately based on detected sentiment.

# Task 6: Multilingual Chatbot

### Objective

Extend the chatbot to support multilingual conversations across multiple languages while preserving context, intent, and conversational continuity.

### Features

* Automatic language detection
* Support for multiple languages
* Mixed-language conversation handling
* Cross-lingual reasoning
* Context preservation
* Intent recognition across languages
* Consistent multilingual responses

### Expected Outcome

A multilingual chatbot capable of understanding and responding intelligently regardless of language while maintaining conversational context.

# Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd AI-Internship
```

Install the required dependencies:

```bash
pip install -r requirements.txt
```

Run any Streamlit application:

```bash
streamlit run app.py
```
